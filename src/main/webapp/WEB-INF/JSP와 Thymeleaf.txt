jsp 파일은 WEB-INF 폴더에 배치하고, Thymeleaf 는 resource 폴더에 위치하는가?

jsp = JavaServer Pages
컨트롤러 없이 뷰에서 모델로 직접 불러올 수 있다.

HTML 파일 안에 자바 코드 섞어서 서버에서 실행할 수 있는 VIEW 기술

예)

index.jsp
<% String name = "홍길동"; %>
<p>안녕하세요, <%=name %> 님

Thymeleaf HTML 파일 안에서 ${변수} 같은 문법으로 데이터 표현하는 템플릿 엔진
JSP 처럼 자바코드가 변환되는 것이 아니라
단순히 HTML 기반 코드만 작성
타임리프의 경우 단순히 HTML 코드만 작성하기 때문에
컨트롤러로 제공받기 때문에 서버나 중요 보안자바코드에 직접 접근할 일 없다
하지만, JSP 의 경우 자바코드에 직접접근 가능해서 폴더보호 같은 보안 필요


MVC = MODEL (디비나 이메일인증변수 등 기능에 필요한 명칭이나 기능작성공간)
      VIEW (사용자가 요청하는 페이지정보를 전달하는 역할. 사람들이 보는 공간)
      CONTROLLER (서버에서 받은 데이터를 DTO 에 받아서 전달 컨트롤러는 중간다리역할)
